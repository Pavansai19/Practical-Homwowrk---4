Unsupervised Learning on Global Cancer Patient Data (2015-2024)¶

This notebook performs an unsupervised learning analysis on the Global Cancer Patient Dataset (2015-2024) to explore patterns in cancer patient data. The techniques applied include:

  • Principal Component Analysis (PCA) and Singular Value Decomposition (SVD)
  • Matrix Completion
  • Anomaly Detection
  • Top 10 Predictors Identification
  • Variable Correlation Heatmap
  • Hierarchical Clustering with Linkage (Optimized)
  • K-Means Clustering

Dataset¶

The dataset contains cancer patient records with:

  • Numerical predictors: Age, Genetic_Risk, Air_Pollution, Alcohol_Use, Smoking, Obesity_Level, Treatment_Cost_USD, Survival_Years
  • Categorical variables: Gender, Country_Region, Cancer_Type, Cancer_Stage

Research Questions¶

  • PCA/SVD: Which risk factors explain the most variance in patient outcomes?
  • Matrix Completion: Can we reconstruct missing data accurately?
  • Anomaly Detection: Are there any anomalous patient records?
  • Top Predictors: What are the top 10 predictors of patient outcomes?
  • Clustering: Can we identify distinct patient risk profiles?

5. Matrix Completion with SVD¶

Simulate missing data and reconstruct it using Singular Value Decomposition (SVD).

In [9]:
# Simulate missing data by introducing 10% NaNs
X_missing = X_num_scaled.copy()
mask = np.random.random(X_missing.shape) < 0.1
X_missing[mask] = np.nan

# SVD-based matrix completion function
def matrix_completion(X, k=3, max_iter=10):
    X_filled = SimpleImputer(strategy='mean').fit_transform(X)
    for _ in range(max_iter):
        U, S, Vt = svd(X_filled, full_matrices=False)
        S_k = np.diag(S[:k])
        X_reconstructed = U[:, :k] @ S_k @ Vt[:k, :]
        X_filled[mask] = X_reconstructed[mask]
    return X_filled

# Perform matrix completion
X_imputed = matrix_completion(X_missing, k=3)

# Calculate Mean Squared Error (MSE) for imputed values
mse = np.mean((X_num_scaled[mask] - X_imputed[mask]) ** 2)
print(f"\nMatrix Completion MSE: {mse:.4f}")
